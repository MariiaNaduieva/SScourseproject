Summer School Course project

Subject: Library

The Book tab contains a table with data about the books in the library. You can also create a book (add a new book), edit an existing book or delete a book.

The Reader tab contains a table with data about readers who have visited the library. You can also create a reader (add a new reader), edit an existing reader or delete a reader.

The Issued Book tab contains a table with data about all the books issued. Also on the page you can create a book (add a new book), edit a book or delete a book.
